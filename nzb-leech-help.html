Nzb Leech Help
Updated January 28 2013

*** Use Chrome, Opera (not Opera Mini) or Firefox to download nzb files as default browser does not work with some nzb sites ***

Nzb Leech allows you to download binaries from usenet.  You must add an usenet server in Settings first.

Once configured you press and hold on the main screen to import any nzb.  The list of nzb files that Nzb Leech picks up are from the default download directory for your device.

Alternatively, you can load a nzb from the built in file manager.

If there are missing segments for the current file, the file will be shown in red text and till not download fully.

The currently downloading file is shown in light blue, and the currently decoding file is shown in a darker blue.

To cancel a download, press the back button on your device or select quit.

Donate Version Extra Features
To resume a download import the nzb file again and Nzb Leech will resume from where it left of.

To queue nzbs select the option and specify the folder to get the next nzb file to download. You still must first start the first nzb download manually.

To pause/resume a download select the menu item.

To switch between WIFI/3G pause your download, wait for your connections to disconnect, turn on/off WIFI, then resume your download. Sometimes you won't be able to connect right away as the server may prevent you from connecting via multiple ips (3G and WIFI)
